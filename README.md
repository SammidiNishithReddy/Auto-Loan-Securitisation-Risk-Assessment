# Auto Loan Securitisation Risk Assessment

## Project Overview
This Power BI project analyzes an Auto Loan Securitisation portfolio and provides insights into:

- Executive Portfolio Performance
- Credit Risk Analytics
- Delinquency & Collections Monitoring
- IFRS 9 Expected Credit Loss (ECL)
- Collections & Recovery Performance
- Vintage Analysis
- Investor Reporting

## Tools Used
- Microsoft Power BI
- DAX
- Data Modeling
- Financial Risk Analytics

## Key Metrics
- Portfolio Outstanding
- Collection Efficiency
- Default Rate
- Recovery Rate
- Expected Credit Loss (ECL)
- Pool Factor
- CPR & SMM
- HHI Concentration Index
